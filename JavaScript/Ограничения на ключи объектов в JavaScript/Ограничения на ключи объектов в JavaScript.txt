№1: 
- Для ключа '1a' кавычки oбязательны, так как он начинается с цифры и не содержит специальных символов.
- Для ключей 'b2' и 'e5' кавычки необязательны.
- Для ключа 'с-с' кавычки обязательны, так как он содержит дефис, который является специальным символом.
- Для ключа 'd 4' кавычки обязательны, так как он содержит пробел, который также является специальным символом.

№2:

let obj = {
  '1a': 1,
  'b2': 2,
  'с_с': 3,
  'd4': 4,
  'e5': 5
};

console.log(obj['1a']);
console.log(obj.b2);
console.log(obj['c-c']);
console.log(obj['d 4']);
console.log(obj.e5);
