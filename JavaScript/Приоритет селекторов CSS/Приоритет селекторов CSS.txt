№1
После выполнения данного кода заголовки примут размер 30px
<h2>text</h2>
h2 {
	font-size: 20px;
}
h2 {
	font-size: 30px;
}

№2
Заголовки будут иметь размер в 20px, заголовки курсивные и цвет будет красным.
<h2>text</h2>
h2 {
	font-size: 20px;
	color: blue;
}
h2 {
	font-style: italic;
	color: red;
}
